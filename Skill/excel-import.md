# Excel 导入

## 使用场景

当需要将 Excel(.xlsx) 文件中的数据批量导入数据库表时使用，例如：

- 从 Excel 文件导入业务数据到数据库
- 批量更新/补充数据库表记录
- 数据迁移或数据初始化

## 执行步骤

### 1. 读取 Excel 信息

使用 `excel_reader` 工具读取 Excel 文件，获取：
- 列名列表
- 前 5 行数据样例
- 总行数
- 每列数据类型推断

### 2. 获取目标表结构

使用 `input_sql` 执行 `DESC <表名>` 获取表结构：
- 字段名、类型、是否可为 NULL

若表不存在，告知用户并终止流程。

### 3. 推断列映射

根据 Excel 列名和表字段名自动推断映射关系：
- 优先精确匹配（忽略大小写）
- 其次模糊匹配（列名包含字段名，或字段名包含列名）
- 无法匹配的列标记为 `[未匹配]`

### 4. 展示并确认映射

向用户展示映射关系，格式：

```
| Excel列名 | → | 数据库字段 | 类型 | 是否可为NULL |
```

- 如有不可为 NULL 的字段未被映射，**必须明确警告用户**
- 等待用户确认，用户可指出需要修改的映射
- **不得擅自修改，必须经用户确认**

### 5. 调用导入执行

用户确认后，调用 `excel_agent_tool(content)`，content 必须包含：
- Excel 文件路径
- 目标表名（含数据库名，如 `database.table`）
- 已确认的列映射关系

示例：
```
将 ./test_data.xlsx 导入到 daijia_coupon.coupon_info 表，列映射为：优惠券类型→coupon_type, 优惠券名字→name, 金额→amount
```

### 6. 反馈结果

将 `excel_agent_tool` 的执行结果（导入成功/失败、影响行数）告知用户。
